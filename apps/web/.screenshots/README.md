# Visual review captures

These images are intentionally committed as pull-request evidence. The project owner requested a self-contained PR because remote review will happen without access to the development device.

- `final-*`: current desktop, mobile, docs, guide, CLI, architecture, and light-theme captures.
- `iter0-*` through `iter6-*`: visual iteration history from scaffold baseline through final polish.

The images are review artifacts, not runtime application assets; the site does not load them.
