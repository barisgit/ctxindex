## Exact command syntax

The executable already renders its command tree from the same Citty definitions that parse invocations. Use `ctxindex <command> --help` instead of a second prose command inventory:

```bash
ctxindex --help
ctxindex search --help
ctxindex source add --help
ctxindex extension catalog --help
```
