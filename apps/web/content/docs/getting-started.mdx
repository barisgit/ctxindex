---
title: Install ctxindex
description: Install the released CLI or run ctxindex from a source checkout.
---

Install the released CLI with the repository-pinned Bun runtime:

```bash
bun add --global ctxindex
ctxindex --help
```

For a source checkout:

```bash
git clone https://github.com/barisgit/ctxindex.git
cd ctxindex
bun install --frozen-lockfile
bun cli --help
```

`bun cli` is the development equivalent of the installed `ctxindex` executable. Both keep configuration, SQLite projections, cached Artifacts, and OAuth Grants on the local machine.

Continue with the [five-command providerless walkthrough](/docs), [connect a mail or calendar Account](/docs/start/connect-provider), or [install the external example Extension](/docs/use/extensions). Use `ctxindex <command> --help` for exact current syntax instead of a second handwritten command inventory.
