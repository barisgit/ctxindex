ctxindex gives shell-capable agents one local typed interface over mail, calendars, files, and Extension-defined context. The CLI is the integration surface: agents compose deterministic commands, read `--json`, and act only through configured Sources.

## Start

Install ctxindex, index one local directory without an Account, then optionally connect Google or Microsoft.
