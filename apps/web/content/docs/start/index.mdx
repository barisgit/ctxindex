---
title: Install and reach first success
description: Install ctxindex, index one local file without an Account, and return a useful result.
---

The shortest trustworthy path starts without OAuth or provider traffic. You will create one Realm, connect a providerless local directory Source, materialize it, and search it.

<Callout>
  ctxindex currently requires Bun 1.3.14. Until the first npm release is available, run the development CLI from a source checkout; after release, the same commands use the global `ctxindex` executable.
</Callout>

<Tabs items={["Released package", "Source checkout"]}>
  <Tab value="Released package">
    ```sh
    bun add --global ctxindex
    ctxindex --help
    ```
  </Tab>
  <Tab value="Source checkout">
    ```sh
    git clone https://github.com/barisgit/ctxindex.git
    cd ctxindex
    bun install
    bun cli --help
    ```

    Replace `ctxindex` with `bun cli` in the remaining commands.
  </Tab>
</Tabs>

## Create local context

<Steps>
  <Step>
    Create one file and initialize ctxindex.

    ```sh
    mkdir -p "$HOME/context"
    printf 'Project Aurora kickoff is Tuesday at 10:00.\nOwner: Maya Chen.\n' \
      > "$HOME/context/aurora.txt"
    ctxindex init
    ```
  </Step>
  <Step>
    Create a Realm and connect the directory as one Source.

    ```sh
    ctxindex realm add work --name "Work"
    ctxindex source add local.directory \
      --realm work \
      --label work-files \
      --config-root-path "$HOME/context"
    ```
  </Step>
  <Step>
    Materialize and search that Source.

    ```sh
    ctxindex sync --source work-files --json
    ctxindex search "Aurora kickoff" \
      --realm work \
      --kind file \
      --local-only \
      --json
    ```
  </Step>
</Steps>

A successful search returns a stable Ref and a bounded result envelope:

```json
{
  "results": [
    {
      "ref": "ctx://01J00000000000000000000000/file/aurora.txt",
      "profile": { "id": "file", "version": 1 },
      "sourceId": "01J00000000000000000000000",
      "origin": "local",
      "title": "aurora.txt"
    }
  ],
  "pagination": { "offset": 0, "limit": 20, "hasMore": false },
  "warnings": []
}
```

Pass the returned Ref through unchanged:

```sh
ctxindex get 'ctx://01J00000000000000000000000/file/aurora.txt' --json
ctxindex export 'ctx://01J00000000000000000000000/file/aurora.txt' --format json
```

Do not reconstruct provider-owned Ref suffixes. Copy the Ref returned by ctxindex.
