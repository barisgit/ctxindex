## Package direction

The public CLI composes private core, Adapter, Profile, and Extension SDK workspaces. Extensions import the public SDK facade and ordinary definition packages; they do not import core runtime internals.

The website projects those contracts without owning product behavior or user/provider state.

## Visual contribution

ctxindex is a quiet precision instrument: neutral slate structure, one rare amber signal, real product artifacts, flat surfaces, and exact typography. Avoid gradients, decorative AI imagery, repeated generic card grids, and terminal cosplay.

Use the live [`/design`](/design) specimen and public [`/brand`](/brand) assets instead of inventing parallel tokens or logo geometry.
