---
title: Development workflow
description: Install the workspace, run focused tasks, and complete repository verification.
---

ctxindex pins Bun 1.3.14 and uses Bun workspaces with Turborepo.

```sh
bun install
bun cli --help
bun dev
```

Prefer package-owned focused commands while iterating:

```sh
bun test <focused-test-file>
bun run --cwd apps/web typecheck
bun run --cwd apps/web build
```

Before completing a non-trivial change:

```sh
bun run ci
bunx openspec validate --all --strict
```

Worktrees created by the repository helper isolate CLI state so one change's tests do not reuse another worktree's database, config, or installed Extension state.

Never commit secrets or `.env` files, delete user data under `~/.local/share/ctxindex` without consent, or test live provider traffic when deterministic fixtures prove the contract.
