# ctxindex CLI

`ctxindex` is a local personal-context gateway for agents. This package installs the Bun-powered `ctxindex` command.

Install Bun 1.3.14, then install the package globally:

```sh
bun add --global ctxindex
ctxindex --help
```

On Linux, native Keychain support also requires `libsecret`. For Debian and
Ubuntu, install it with `sudo apt-get install libsecret-1-dev` before installing
ctxindex.

The project is pre-alpha. See <https://github.com/barisgit/ctxindex> for usage, trust boundaries, and contributor documentation.
