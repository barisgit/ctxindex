# ctxindex bundled skills

This directory holds agent-facing skill docs that travel with the ctxindex release.

Agents and CLI users discover and read these via:

```bash
ctxindex skills list
ctxindex skills get <name>
ctxindex skills path
```

The skill set is intentionally minimal until the CLI surface stabilizes. See `SPEC.md` section "Skills surface" and `IMPLEMENTATION.md` section "CLI surface and output" for normative behavior.
